# 语义分割分析工具

一个用于管理和分析语义分割实验的轻量仓库骨架，面向模型对比、结果整理和实验资产管理。

## 项目目标

- 统一管理不同语义分割模型的权重文件
- 支持后续扩展评估、可视化和结果对比脚本
- 将本地数据集、模型权重与代码仓库解耦，避免大文件直接进入 Git 历史

- `models/` 中的 `.pth` 权重文件体积较大，默认不纳入版本管理。

## 适用场景

- 比较不同语义分割模型在同一数据集上的表现
- 管理 clean / adv 等不同训练设置对应的 checkpoint
- 为后续加入评估脚本、可视化脚本和实验报告打基础

## 建议后续补充

- `src/`：核心分析与评估代码
- `configs/`：实验配置
- `results/`：指标、图表和导出结果
- `notebooks/`：交互式分析记录

## 大文件管理说明

当前仓库默认忽略模型权重和数据集目录，避免普通 GitHub 仓库因大文件上传失败。

如果后续确实需要托管权重，建议使用以下方案之一：

1. Git LFS
2. Hugging Face Hub
3. 独立网盘或对象存储，并在仓库中提供下载说明
